## Setup

### Docker

    docker-compose up

Open [localhost:4000](http://localhost:4000) to view the page.


### Local Setup

Install [rvm](https://rvm.io/)

    rvm install 2.2
    rvm --default 2.2
    ruby -v
    # Install dependency manager
    gem install bundler

    
Install ruby dependencies

    cd <project-dir>
    bundler
    
Run Project

    jekyll serve
